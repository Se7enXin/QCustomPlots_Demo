# QCustomPlot-Demo

## Installation

```sh
& git clone https://github.com/Se7enXin/QCustomPlots_Demo.git
& cd QCustomPlots_Demo/source
& qmake -project
& qmake
& make

```

## Usage

```sh
& ./run.sh

```
